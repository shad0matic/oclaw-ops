# Keyboard Shortcuts

## Command Palette

Quick access to navigation, agents, and tasks.

### Opening the Command Palette

| Shortcut | Platform | Notes |
|----------|----------|-------|
| `Ctrl + K` | Mac/Linux | Standard shortcut |
| `Cmd + K` | Mac | Standard shortcut |
| `Ctrl + /` | **Windows** | Use this if `Ctrl+K` opens browser search |
| `Ctrl + P` | All | VS Code-style alternative |

### Using the Command Palette

Once open:
- **Type to search** — filters pages, agents, and tasks
- **Arrow keys** — navigate results
- **Enter** — select/navigate
- **Escape** — close

### Available Commands

**Pages**
- Overview
- Lab
- Kanban
- Knowledge
- Costs
- Agents
- Events
- Settings

**Dynamic**
- All registered agents (navigates to agent detail)
- All tasks (navigates to task detail)
